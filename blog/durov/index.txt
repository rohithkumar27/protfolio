e:T1150,<h2>1. core philosophy: freedom first</h2>
<ul>
<li>freedom over comfort
<ul>
<li>freedom of action matters more than convenience, popularity, or approval</li>
</ul>
</li>
<li>independence is structural
<ul>
<li>real freedom comes from removing dependencies, not negotiating with them</li>
</ul>
</li>
<li>principles > short-term wins
<ul>
<li>compromise compounds; once crossed, lines cannot be redrawn</li>
</ul>
</li>

<ul>
<li>the ability to walk away preserves autonomy</li>
</ul>
</li>
</ul>
<h2>2. money, success, and fame</h2>
<ul>
<li>money is a tool, not a goal
<ul>
<li>wealth attracts pressure from states, investors, and public opinion</li>
</ul>
</li>
<li>no ads, no shareholders
<ul>
<li>every external stakeholder becomes a potential lever of control</li>
</ul>
</li>
<li>skeptical of fame
<ul>
<li>attention distorts incentives and decision-making</li>
</ul>
</li>
<li>optionality > valuation
<ul>
<li>maximal scale can trap founders more than it frees them</li>
</ul>
</li>
</ul>
<h2>3. work, creativity, and building</h2>
<ul>
<li>small teams outperform large ones
<ul>
<li>coordination cost grows faster than productivity</li>
</ul>
</li>
<li>automation over headcount
<ul>
<li>more people often create internal friction, not better products</li>
</ul>
</li>
<li>deep, uninterrupted work
<ul>
<li>originality requires silence and long focus windows</li>
</ul>
</li>
<li>products over narratives
<ul>
<li>real utility spreads without persuasion or hype</li>
</ul>
</li>
</ul>

<ul>
<li>not a feature toggle or marketing claim</li>
</ul>
</li>
<li>no backdoors, ever
<ul>
<li>any “lawful access” becomes a universal vulnerability</li>
</ul>
</li>
<li>security must be technical
<ul>
<li>promises are meaningless without architectural guarantees</li>
</ul>
</li>
<li>integrity over market access
<ul>
<li>better to exit a country than weaken user protections</li>
</ul>
</li>
</ul>

<ul>
<li>censorship now arrives via ids, age checks, and content scanning</li>
</ul>
</li>
<li>political neutrality is non-negotiable
<ul>
<li>once a platform chooses sides, trust collapses</li>
</ul>
</li>
<li>the free internet is fragile
<ul>
<li>openness erodes quietly, step by step</li>
</ul>
</li>
</ul>

</ul>
</li>
